Video2Gif

Motivation

Funny gif can be entertaining, but gif is also useful when creating demo or tutorials. It can demonstrate more information in few seconds. When I am creating my portfolio, I was looking for a free video to gif converter to create demo pieces, but the ones I found don't produce a high-quality gif and it takes a long time to upload when the video file is large.

Therefore, I created my own video to gif converter Video2Gif. It's a desktop application for Mac so I don't have to upload the video. It produces high-quality gif and also allows me to preview the output gif, which is so useful if I am cutting the input video to specific time range for the gif.


Screenshots:

Easy to use, just drag and drop your video to get started
Video2Gif allows you to set start time and end time, and use frames between those time to create the gif. You can see how your gif looks like in the preview window.

How does Video2Gif work

(1) Drag and drop your video
(2) Set start time and end time to cut the video
(3) Click convert, then click on the notification to open output gif
